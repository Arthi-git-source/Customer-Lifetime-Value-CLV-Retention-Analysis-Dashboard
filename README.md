# Customer Lifetime Value (CLV) & Retention Analysis Dashboard

##  Overview

This project analyzes customer behavior to estimate Customer Lifetime Value (CLV), identify churn patterns, and improve retention strategies. It helps businesses make data-driven decisions to maximize revenue and customer engagement.

---

##  Objectives

* Calculate Customer Lifetime Value (CLV)
* Analyze customer retention and churn trends
* Segment customers based on value
* Provide actionable insights for improving retention

---

##  Tools & Technologies

* Python (Pandas, NumPy)
* SQL
* Power BI
* Excel

---

##  Key Features

* Customer segmentation (high, medium, low value)
* Retention & churn analysis
* KPI-based dashboard (CLV, retention rate, churn rate)
* Interactive Power BI visualizations

---

## 📁 Project Structure

* `/data` → Raw dataset (excluded from repo)
* `/notebooks` → Data analysis (Python/Jupyter)
* `/sql` → SQL queries
* `/dashboard` → Power BI dashboard file
* `/images` → Dashboard screenshots

---

##  Key Insights

* Identified high-value customers contributing majority of revenue
* Detected churn-prone segments for targeted retention strategies
* Provided insights to improve customer engagement and profitability

---


##  Business Impact

This project enables businesses to focus on high-value customers, reduce churn, and optimize long-term revenue through data-driven strategies.
